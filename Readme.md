## pubspec.yaml file

#### Add the following to the "pubspec.yaml" file:
```Java
  assets:
    - assets/cars/
    - assets/fonts/Dogs.ttf

  fonts:
    - family: Dogs
      fonts:
      - asset: assets/fonts/Dogs.ttf
```
